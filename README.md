Cinema Quest

A responsive movie browsing website built using HTML, CSS, and JavaScript.
It provides a modern and user-friendly interface to explore movies with attractive UI design.

🚀 Features
Responsive design for all devices
Modern UI/UX layout
Movie cards and posters display
Smooth navigation
Built using pure HTML, CSS, and JavaScript

Technologies Used
HTML5
CSS3
JavaScript
